# Ringvorlesung Digital Humanities: TextLab

Vorbereitungsmaterialien für eine 90-minütige Vorlesung zur Vorstellung von TextLab in der Ringvorlesung Digital Humanities an der LMU.

## Struktur

- `index.qmd`: knappe Projekt-/Materialseite
- `slides/textlab-ringvorlesung.qmd`: Hauptfolien
- `assets/`: CSS, Bilder und vorläufige TextLab-Screenshots aus dem IDK-Workshop

## Render

```bash
quarto render
```

Gerenderte Ausgabe landet in `_site/` und wird nicht versioniert.
